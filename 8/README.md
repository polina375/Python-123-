. Описание предметной области
Приложение предназначено для отслеживания курсов валют и управления подписками пользователей. Были реализованы следующие сущности:

2.1. Модель Author (Автор)
name — имя автора приложения

group — учебная группа автора

2.2. Модель App (Приложение)
name — название приложени
version — версия приложения
author — ссылка на объект Author

2.3. Модель User (Пользователь)
id — уникальный идентификатор пользователя

name — имя пользователя

2.4. Модель Currency (Валюта)
id — уникальный идентификатор валюты

num_code — цифровой код валюты (3 цифры)

char_code — символьный код валюты (3 символа)

name — название валюты

value — текущий курс к рублю

nominal — номинал (количество единиц валюты)

2.5. Модель UserCurrency (Подписка)
id — уникальный идентификатор подписки

user_id — идентификатор пользователя

currency_id — идентификатор валюты

Связи между сущностями:

Один пользователь может быть подписан на несколько валют (связь "многие ко многим")

Одна валюта может быть в подписках у нескольких пользователей

Связь реализована через промежуточную сущность UserCurrency
